## ‚ùì FAQ

### General Questions

**Q: Do I need to create an account?**
> No! Everything saves to your browser's local storage. No login required.

**Q: Will my plans be saved if I close the browser?**
> Yes! Plans are saved to local storage and persist between sessions.

**Q: Can others see my plans?**
> Only if you share the link. Plans are private by default.

**Q: Does this connect to Last War servers?**
> No. This is a standalone planning tool. It doesn't access game data.

### Building & Placement

**Q: How do I rotate a building?**
> Click the building, then use the rotation buttons (0¬∞, 90¬∞, 180¬∞, 270¬∞) in the Property Panel.

**Q: Can buildings overlap?**
> No, buildings snap to grid and cannot occupy the same tiles.

**Q: How do I delete all buildings?**
> Click the üóëÔ∏è Clear button in the top toolbar.

**Q: What's the grid size?**
> 50√ó50 tiles, representing your hive area.

### Export & Sharing

**Q: What is Origin X/Y?**
> The coordinates where your hive's top-left corner (0,0) maps to on the game's world map. Set this to get accurate game coordinates in CSV export.

**Q: How do I find my hive's coordinates?**
> In Last War, open World Map, tap your Marshal's base, note the X:Y coordinates shown.

**Q: Can someone edit my plan from a shared link?**
> No. They can only VIEW and COPY to their own account.

**Q: What's the difference between PNG, CSV, and JSON?**
> - **PNG**: Image file for sharing visually
> - **CSV**: Spreadsheet with coordinates for each player
> - **JSON**: Complete data backup for import/restore

### Pro Features

## üîß Troubleshooting

### Problem: Buildings won't place

**Solution:**
1. Make sure a building is selected (shows ‚úì)
2. Click directly on the grid area
3. Ensure you're not in Pan (‚úã) or Delete (üóëÔ∏è) mode

### Problem: Can't see my saved plan

**Solution:**
1. Click ‚ò∞ Menu
2. Check "ALL PLANS" list
3. Click the plan name to load it
4. If missing, try importing from JSON backup

### Problem: CSV shows wrong coordinates

**Solution:**
1. Check Origin X and Origin Y values
2. These should match your actual hive location in-game
3. Re-export after correcting

### Problem: Zoom is too far/close

**Solution:**
1. Use + / - buttons (bottom right)
2. Click ‚åÇ (Home) to reset view
3. Mouse wheel also zooms

### Problem: Lost my plan

**Solution:**
1. If you exported JSON before, click ‚ò∞ Menu ‚Üí Import JSON
2. If you shared a link, open it to recover
3. Browser data clear = plans lost (export backups!)
